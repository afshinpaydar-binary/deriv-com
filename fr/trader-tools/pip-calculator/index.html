Comment calculer la valeur du pip

La valeur du pip pour un contrat sur DMT5 est calculée en fonction de cette formule:

Valeur du pip = valeur en points × volume × taille du contrat

Pour les comptes synthétiques, la valeur du pip est calculée en USD.

Pour les comptes financiers, la valeur du pip est dans la devise de cotation pour les paires de devises.

Exemple de calcul

Valeur du pip pour les indices synthétiques

Supposons que vous souhaitiez trader 1 lot d'indice de volatilité 75.

  1. La valeur en points est dérivée des chiffres actuels des actifs. Dans l'exemple, le chiffre est 2, donc la valeur du point est 0,01.
  2. La taille du contrat est un lot standard d'indice de Volatilité 75 = 1

La valeur de votre pip est donc 0,01 USD.

Valeur du pip pour le forex

Supposons que vous souhaitiez trader 2 lots d'EUR/USD.

  1. La valeur en points est dérivée des chiffres actuels des actifs. Dans l'exemple, le chiffre est 5, donc la valeur en points est 0,00001.
  2. Un lot standard de forex = 100 000 unités

La valeur de votre pip est donc 2 USD.
